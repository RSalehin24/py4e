## Tuple

### Declaration
> Immutable and irreversible
```python
tuple_example = ('a', 'b', 'c', 'd')
```

### Assignment
> tuples can be assigned to tuples
```python
(a, b, c) = (1, 2, 3)
(a, b, c) = function()
# function returns a tuple
```

### Comparison between tuples
> compares values from the left to the right, if the values are the same then moves to the next value if values are not same then compares and returns true or false
```python
(2, 4, 7) < (3, 5, 9)
```
